# Node Web Application

This project is a simple full-stack activity project with:

- React frontend in `frontend`
- Node.js and Express backend in `backend`

## Run from the project root

```powershell
npm run backend
```

```powershell
npm run frontend
```

## Run the backend

```powershell
cd backend
npm install
npm start
```

## Run the frontend

```powershell
cd frontend
npm install
npm run dev
```

The React app runs on `http://localhost:5173` and fetches data from the Express API at `http://localhost:5000/api/message`.
